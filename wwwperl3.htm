とほほのperl入門（利用例編）

トップ > とほほのperl入門 > 利用例編

■ 制御

◆ 条件判断する 
if ($xx > 100) {
    print "大きい\n";
} else {
    print "小さい\n";
}
◆ 10回処理を繰り返す 
for (my $i = 0; $i < 10; $i++) {
    print "$i\n";
}

■ 文字列

◆ 文字を1文字ずつ処理する 
my $str = "ABCDEFG";
for (my $i = 0; $i < length($str); $i++) {
    my $c = substr($str, $i, 1);
    print "[$c] ";
}
◆ 文字を1文字ずつ処理する(日本語対応) 
use Encode;

my $str = decode('utf8', "ABCDEFGあいうえお");   # ファイルの文字コードは utf8とする.
for (my $i = 0; $i < length($str); $i++) {
    my $c = substr($str, $i, 1);
    print encode('utf8', "[$c] ");
}
◆ 行末の改行コードや空白を取り去る 
$str =~ s/\s*$//;
◆ 全角文字が含まれているか調べる 
use Unicode::EastAsianWidth; # 要インストール
# $strの漢字コードを utf-8にしておいて
if ($str =~ /\p{InFullWidth}/) { print "全角文字が含まれてます"; }

■ 書き出し

◆ 文字列を書き出す 
print "あいうえお\nかきくけこ\n";
◆ ダブルクォーテーションを含む文字列を書き出す 
print "<a href=\"index.html\">\n";
print '<a href="index.html">';
◆ アットマークを含む文字列を書き出す 
print "foo\@abc.xyz";
print 'foo@abc.xyz';
◆ 書式を指定して書き出す 
printf("%04d/%02d/%02d\n", $year, $month, $mday);
◆ 数行にわたる内容を書き出す 
print <<END_OF_DATA;
あいうえお
かきくけこ
END_OF_DATA
◆ バッファをフラッシュする 
$old = select(OUT); $| = 1; select($old);
◆ ファイルをロックする 
open my $fh, '>', "file.txt" or die "Can't open file.txt";
flock $fh, 2;
print $fh ".......\n";
close $fh;  # closeすると自動的にアンロックされる

■ 配列

◆ 配列を用いる 
my @xx;
$xx[0] = "あああ";
$xx[1] = "いいい";
print "$xx[0]\n";
◆ すべての配列要素に対して処理する 
for my $yy (@xx) {
    print $yy;
}

■ ファイル

◆ ファイルを読み出す 
open(IN, "xxxx.txt");
open my $fh, '<', "xxxx.txt" or die "Can't open xxxx.txt";
while (<$fh>) {
    print $_;
}
close $fh;
◆ ファイルのサイズを得る 
$size = -s $file;
◆ ファイルをバイナリモードで読み出す 
my $size = -s $file;
open $fh, '<', $file or die "Can't open $file";
binmode $fh;              # Windowsで必要
sysread $fh, my $buf, $size;
close $fh;
◆ ファイルが存在するか確認する 
if (-f $file) {
    print "ファイルは存在します。\n";
}
◆ ファイルをロックする 
open my $fh, '>>', "outfile.txt" or die "Can't open outfile.txt";
flock $fh, 2;
truncate $fh, 0;
print $fh "....";
close $fh;

■ サブルーチン

◆ サブルーチンを定義する 
sub add {
    return($_[0] + $_[1]);
}
◆ サブルーチンを呼び出す 
$answer = add(3, 5);
◆ サブルーチンの中で使う変数をローカル変数とする 
sub add {
    my ($arg1, $arg2) = @_;
    return $arg1 + $arg2;
}

■ その他

◆ コマンドライン引数を解釈する 
foreach my $arg (@ARGV) {
    print "$arg\n";
}
◆ 時間帯を日本時間に設定する 
$ENV{'TZ'} = "JST-9";
◆ プログラム強制中断の際の処理を記述する 
$SIG{'INT'} = $SIG{'HUP'} = $SIG{'QUIT'} = $SIG{'TERM'} = \&sigexit;
sub sigexit {
    # この部分に作業ファイル削除などの終了処理を記述する
}
